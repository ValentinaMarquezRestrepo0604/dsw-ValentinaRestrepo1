La interfaz de usuario e Interacción Humano-Ordenador



Propósito:

Este sitio Web tiene como propósito que el usuario pueda comunicarse con la máquina de manera intuitiva , teniendo clara la funcionalidad y el manejo del sistema y los procesos, para que exista una afinidad entre el usuario, el producto, las tareas y los procesos. Por otro lado, se quiere que se tenga un enfoque de desarrollo de interfaces y sistemas interactivos para que las personas puedan conectarse con las máquinas y se tengan resultados óptimos.





INTRODUCCIÓN



La interfaz de Usuario

Una interfaz de usuario es el medio por el cual una persona controla una aplicación de software o dispositivo de hardware. Por ejemplo, los sistemas operativos Macintosh y Windows tienen diferentes interfaces de usuario y comparten muchos de los mismos elementos, como un escritorio, ventanas o iconos. Una interfaz de usuario consta de dos componentes: el lenguaje de presentación, que se refiere a la transacción de la computadora hacia la persona, y el lenguaje de acción, que se caracteriza por ser la interacción de la persona hacia la computadora.

Interacción Humano Ordenador

La Interacción Humano-Computadora estudia cómo las personas interactúan con las computadoras y hasta qué punto las computadoras se desarrollan para interactuar con las personas. HCI consta de tres componentes: los usuarios, los ordenadores y la interacción entre ellos. Este dominio interdisciplinario abarca disciplinas como; la informática, la ingeniería de los factores humanos,la inteligencia artificial, la lingüística, la filosofía, la antropología y la ciencia cognitiva.
