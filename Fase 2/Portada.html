La interfaz de usuario e Interacción Humano Ordenador

Este sitio Web tiene como propósito que el usuario pueda comunicarse con la máquina de manera intuitiva , teniendo clara la funcionalidad y el manejo del sistema y los procesos, para que exista una afinidad entre el usuario, el producto, las tareas y los procesos. Por otro lado, se quiere que se tenga un enfoque de desarrollo de interfaces y sistemas interactivos para que las personas puedan conectarse con las máquinas y se tengan resultados óptimos.

la interfaz de Usuario

Una interfaz de usuario es el medio por el cual una persona controla una aplicación de software o dispositivo de hardware.
Por ejemplo, los sistemas operativos Macintosh y Windows tienen diferentes interfaces de usuario y comparten muchos de los
mismos elementos, como un escritorio, ventanas o iconos. Una interfaz de usuario consta de dos componentes: el lenguaje de
presentación, que se refiere a la transacción de la computadora hacia la persona, y el lenguaje de acción, que se caracteriza
por ser la interacción de la persona hacia la computadora. Es fundamental un buen sitio Web para que las empresas puedan
incrementar sus ventas y su crecimiento, satisfaciendo las necesidades del usuario ya que la tecnología esta siempre
presente en nuestras vidas. Por lo tanto, siempre se deben comprender las exigencias y las necesidades de los clientes y
la facilidad a la hora de navegar para que el usuario se sienta cómodo, para permanecer en la competencia.

Interacción Humano Ordenador

La Interacción Humano-Computadora estudia cómo las personas interactúan con las computadoras y hasta qué
punto las computadoras se desarrollan para interactuar con las personas. HCI consta de tres componentes:
los usuarios, los ordenadores y la interacción entre ellos.
Este dominio interdisciplinario abarca disciplinas como; la informática, la ingeniería de los factores humanos,
la inteligencia artificial, la lingüística, la filosofía, la antropología y la ciencia cognitiva.
Se trata de hacer que los sistemas sean fáciles de aprender y usar, con el propósito de mejorar la competitividad,
confiabilidad en los clientes y reduce los costes.
Para garantizarlo se deben realizar pruebas de usabilidad en donde el equipo establece los aspectos positivos y negativos
del prototipo antes de seguir desarrollándolo permitiéndole a los equipos tener una imagen detallada de cómo funciona el sistema.
